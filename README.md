# craw_neau

## 工具 
- PyCharm
- Anaconda 4.5.12
- MySQL 5.5


## 环境  
- 学校内网
- python 3.6


## 实现功能 
- 自动识别图片验证码登录教务处
- 获取当前登录账号信息存入数据库，获取个人证件照存入图片服务器
- 获取当前（历史）成绩
- 获取个人当前（历史）课表
